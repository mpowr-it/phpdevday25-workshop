# COMMANDS

```text
- Use tmux command (control+b) <arrow-up>|<arrow-down> to activate switch between panes
- All panes have the `remain-on-exit` setting activated by default.
- When you exit out of the shell or app, the panes will remain.
- Use tmux command (control+b) `:kill-pane` to remove the pane.
- Use tmux command (control+b) `:respawn-pane` to restart the shell in the pane.
```
